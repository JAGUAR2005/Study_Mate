# StudyMate AI — Project Charter

## Purpose

StudyMate AI helps students turn difficult academic PDF passages into personalized study material quickly. A learner uploads a PDF, selects confusing text, generates an AI action, edits the result, and saves it for later review.

## Core hypothesis

If learners can move directly from a confusing passage to editable, source-linked study material, they will understand and retain dense material more effectively than through copy-pasting between separate tools.

## Primary user

Students and independent learners reading technical or academic PDFs on a desktop browser.

## MVP outcome

One complete path must work reliably:

`upload PDF → read/select text → generate study material → edit → save → review`

## MVP scope

- PDF upload and browser reading
- Text selection with a contextual action menu
- Explanation, notes, example, mnemonic, quiz, and visual generation
- Editable study workspace with source/page metadata
- Saved notes, tags, search/filter, and cloze-style review
- Regeneration that preserves user-authored content

## Explicit non-goals for the MVP

- Collaboration or social sharing
- Audio generation
- Native mobile applications
- EPUB or batch multi-file support
- Fully automated OCR for scanned PDFs
- Medical, legal, financial, or other high-stakes tutoring claims

## Success criteria

- A new user reaches a first saved note in under two minutes.
- Every generated item remains linked to its source passage and page.
- Users can edit or delete AI output without losing their own notes.
- The demo can be run by a judge using documented steps and sample data.
